# Testing Unified Numbering

This is a test post to verify that margin notes and footnotes share a unified counter.

First, let's add a margin note.<MarginNote>This should be numbered 1.</MarginNote> The number should appear as "1".

Now let's add a footnote.<Footnote>This should be numbered 2.</Footnote> It should continue the sequence.

Another margin note with an anchor.<MarginNote anchor="this highlighted text">This should be numbered 3.</MarginNote> Notice how it highlights the text.

And another footnote.<Footnote>This should be numbered 4, continuing the sequence from margin notes and footnotes.</Footnote>

One more margin note.<MarginNote>Number 5 here!</MarginNote>

The sequence should be: 1, 2, 3, 4, 5 - unified across both component types.

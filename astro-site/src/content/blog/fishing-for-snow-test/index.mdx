## What Happens During Build

When you import an image from `src/assets/`, Astro:

1. **Detects dimensions automatically** - No need to specify width/height
2. **Generates multiple sizes**: 400w (mobile), 800w (tablet), 1200w (desktop)
3. **Converts to modern formats**: AVIF (70% smaller), WebP (40% smaller), JPG (fallback)
4. **Creates responsive srcset**: Browser picks best size
5. **Optimizes & compresses**: Smart compression for each format
6. **Lazy loads**: Only loads when scrolled into view
7. **Prevents layout shift**: Width/height prevent page jumping

All automatically!
